## شروط الإطلاق الإلزامية

**صفر خلل P0** في: تسجيل الدخول · العزل بين التجار · الدفع المكرر · إنشاء الطلب · القبول والرفض · الاسترجاع · تحديث الحالة · وصول العميل · تأكيد التسليم · إشعارات الفرع · تسوية الأموال · Audit Logs · النسخ الاحتياطي · حماية بيانات الموقع · سياسات الخصوصية.

**معايير القبول النهائية:** P0 = 0 · كل المسارات المالية مختبرة · كل حالات الطلب قابلة للتتبع · **لا صفحات وهمية، لا أزرار غير مربوطة، لا APIs يتيمة، لا جداول غير مستخدمة** · جميع الواجهات متصلة بالـBackend الحقيقي · Staging مطابق للإنتاج · خطة Rollback جاهزة.

## الطيار (المرحلة 15) — مقاييس القرار

نسبة جاهزية الطلب قبل الوصول، زمن الانتظار بعد الوصول، نسبة الاكتشاف الآلي مقابل الزر اليدوي، No-show، نجاح الدفع، رضا الفرع (استبيان أسبوعي). أهدافها تُثبَّت من أول أسبوعي بيانات وتحكم قرار التوسع.

## ما بعد الإطلاق

نسخ v2 المؤجلة (01§2) تُفتح فقط بعد استقرار المؤشرات وقرار موثق في ذاكرة المشروع.
